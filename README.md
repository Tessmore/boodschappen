# Boodschappen

A simple app for keeping track of grocery items and tasks — add things to a list, check them off, and share the list via a hash-based link.

## Developing

Once you've created a project and installed dependencies with `yarn install`

```sh
yarn dev
```

## Building

To create a production version of your app:

```sh
yarn build
```

You can preview the production build with `npm run preview`.
